На экран выведены последние пять операций, отсортированные по дате.
Операции разделены пустой строкой.
Каждая операция на первой строке выводит дату и время операции и тип операции (перевод со счета на счет, открытие вклада и т. п.).
На второй строке выводит частично замаскированный номер, с которого происходит перевод, и частично замаскированный номер, на который происходит перевод.
На третьей строке выведена сумма и валюта.
